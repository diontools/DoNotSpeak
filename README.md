# DoNotSpeak

スピーカーで音楽を再生させないアプリ

## 動作環境

Android 5.0 Lollipop 以降

## 概要

イヤホンを接続していないときに、次のイベントでスピーカーの音量をゼロにする。

* アプリ起動時
* 音量ボタンなどで音量を変更したとき
* イヤホンとの接続を解除したとき
